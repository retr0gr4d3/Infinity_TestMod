# Alpha Testing Tools / Mod Menu
<b> Uses MelonLoader (Mono). Precompiled versions are unavailable at this time, please compile using VS.
Features:

- [X] Client-side FakeDev
- [X] Loaders:
  - [x] Shop
  - [x] Merge
  - [x] Quests (+ Abandon Quests)
- [x] Autoattack + Config
- [x] Packet Tools:
  - [x] Interceptor (halt communications)
  - [x] Sniffer (log Server + Client packets)
  - [x] Sender (send commands to the server)
- [ ] More to come...
